关于

你好，我是 1ux4ry。

目前长期专注在信息安全方向，主要关注 Web 安全、渗透测试、红蓝对抗、漏洞复现和日常代码审计，也会把自己在攻防演练、靶场实战和学习过程里的方法论慢慢沉淀到这个博客里。

我更偏爱“实战驱动学习”这条路线。相比只停留在概念层面，我更喜欢把一个问题拆到足够细，弄清楚它为什么会发生、怎么被利用、怎么被检测、又该怎么修。也正因为这样，这个博客会持续记录我在 Java 安全、内存马、反序列化、攻防演练和靶场复盘里的学习过程。

过往经历里，我参与过教育行业、省级和国家级攻防演练，做过蓝队监测、告警分析、攻击行为研判，也做过红队渗透测试、漏洞利用、权限获取和信息收集。对我来说，攻和防并不是割裂的两件事，而是同一条链路上的两个视角。

我希望这里不只是一个“文章仓库”，更像一份持续更新的安全研究档案。以后会继续把踩过的坑、复盘过的链路、好用的思路和能落地的经验都整理进来，方便自己回看，也希望能帮到同样在路上的你。
