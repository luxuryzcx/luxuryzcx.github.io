import string

def encode(string,string2):
    tmp_str = str()
    ret = str()
    bit_string_str = string.encode()
    remain = len( string ) % 3
    remain_str = str()
    for char in bit_string_str:
        b_char = (bin(char)[2:])
        b_char = '0'*(8-len(b_char)) + b_char
        tmp_str += b_char
    for i in range(len(tmp_str)//6):
        temp_nub = int(tmp_str[i*6:6*(i+1)],2)
        ret += string2[temp_nub]
    if remain==2:
        remain_str = tmp_str[-4:] + '0'*2
        temp_nub = int(remain_str,2)
        ret += string2[temp_nub] + "="
    elif remain==1:
        remain_str = tmp_str[-2:] + '0'*4
        temp_nub = int(remain_str,2)
        ret += string2[temp_nub] + "="*2
    return ret.replace("=","")

  • 回到题目本身我们我肯可以得到s的值，其中s的值都是s_box中取的，因此[b1%31]是可以得知的。也就是
s_box = 'qwertyuiopasdfghjkzxcvb123456#$'

for i in s:
    s_box.index(i)

那么求b1就是

s_box = 'qwertyuiopasdfghjkzxcvb123456#$'

for i in s:

但是要爆破k呢，因为b1是ASCII，最大不超过128，128/31==4因此爆破k值最大不超过5.,因为我们是从0开始爆破，因为b1是从大每次整除31，依次减小的，因此我们s选好 是需要逆序。因此：

import libnum

s_box = "qwertyuiopasdfghjkzxcvb123456#$"
s = "u#k4ggia61egegzjuqz12jhfspfkay"[::-1]

for k in range(31):
    b1 = k
    for i in s:
        b1 = b1 * 31 + s_box.index(i)
    flag = libnum.n2s(int(b1))
    if b"NSSCTF" in flag:
        print(flag)

分析一波

  1. 函数签名：
  • __int64 __fastcall NSS(const void *a1)
  • __fastcall 是一种调用约定，表示参数通过寄存器传递而不是栈。
  • a1 是一个指向 void 类型的指针，表示输入数据的地址。
  1. 局部变量：
  • int v1; 和 int v2;：这两个变量用于临时存储中间结果。
  • int v4[11];：这是一个包含 11 个整数的数组，用于存储从 a1 复制的数据。
  • int i;：这是一个循环计数器。
  • int v3[11];：假设这是一个已知的正确数据数组，用于与 v4 进行比较。
  1. 数据复制：

  • memcpy(v4, a1, sizeof(v4));

  • 将 a1 指向的数据复制到 v4 数组中。

  1. 位操作：

  • v4[0] ^= 0xAu;

  • 将 v4[0] 与 0xA 进行异或操作。

  • v1 = v4[1];

  • 将 v4[1] 的值赋给 v1。

  • LOBYTE(v1) = LOBYTE(v4[1]) ^ 0xBB;

  • 将 v1 的最低字节与 0xBB 进行异或操作。

  • v4[1] = v1;

  • 将 v1 的值赋回 v4[1]。

  • 类似的操作对 v4[2] 到 v4[10] 进行，每个元素都与不同的常量进行异或操作。

  1. 比较：

  • for ( i = 0; i <= 10; ++i )

  • 遍历 v4 数组。

  • if ( v3[i] != v4[i] )

  • 如果 v4[i] 与 v3[i] 不相等，则返回 0i64。

  • return 1i64;

  • 如果所有元素都相等，则返回 1i64。

具体步骤

  1. 数据复制：
  • 将 a1 指向的数据复制到 v4 数组中。
  1. 位操作：
  • 对 v4 数组中的每个元素进行特定的异或操作。
  1. 比较：
  • 将处理后的 v4 数组与已知的正确数据数组 v3 进行逐元素比较。
  • 如果有任何一个元素不相等，则返回 0i64，表示验证失败。
  • 如果所有元素都相等，则返回 1i64，表示验证成功。

    mark(maze, pos)
