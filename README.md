# Hugo 1ux4ry Blog

这是一个按 Hugo 目录结构搭好的个人博客模板。

特点：

- 首页布局参考了你给的博客截图
- 保留顶部导航、居中头像简介、纵向文章流和右侧标签感
- 视觉细节改成更适合个人品牌的暖白 + 橙蓝点缀风格

## 目录

- `config.toml`: 站点配置
- `content/`: 页面和文章内容
- `layouts/`: Hugo 模板
- `static/css/main.css`: 样式
- `static/images/avatar-mark.svg`: 默认头像图形

## 本地运行

如果本机已经安装 Hugo：

```bash
hugo server -D
```

如果还没安装 Hugo，需要先安装 Hugo Extended，再在本目录运行上面的命令。
