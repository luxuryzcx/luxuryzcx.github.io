分析

- 如果索引 `i` 是奇数，那么它会将 `ptr[i]` 左移 `i` 位，然后将其作为无符号整数打印出来。
- 如果索引 `i` 是偶数，那么它会将 `i` 乘以 `ptr[i]` 的值，然后将其作为无符号整数打印出来

我们写一个脚本

```
encrypted = [
    198, 232, 816, 200, 1536, 300, 6144, 984, 
    51200, 570, 92160, 1200, 565248, 756, 
    1474560, 800, 6291456, 1782, 65536000
]

# 创建一个空列表来存储解密后的字符
flag = [''] * 20

# 解密过程
for i in range(1, 20):
    if i & 1:  # 奇数位
        flag[i-1] = chr(encrypted[i-1] >> i)
    else:  # 偶数位
        flag[i-1] = chr(encrypted[i-1] // i)
result = ''.join(flag)
print(result)
```
